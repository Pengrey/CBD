### Initiate db

```bash
mongod -dbpath <directory to be used by mongo>
```

### Initiate mongo shell for iteration with db with cli

```bash
mongosh
```

or

```bash
mongo
```

### Create database

```bash
use <new DB name>
```

### Show databases present

```bash
show dbs
```

### Drop database

```bash
db.dropDatabase()
```

### Create Collection

```bash
db.createCollection(<name>, <options>)
```

### Show collections present on db

```bash
show collections
```

### Drop collection from db

```bash
db.<collection name>.drop()
```

### Insert document into collection

```bash
db.<collection name>.insert(document)
```

### Find documents

```bash
db.<collection name>.find()
```

or 

```bash
db.<collection name>.find().pretty()
```

### Update Document

```bash
db.<collection name>.update(<selection criteria>, <updated data>)
```

### Delete Document

```bash
db.<collection name>.remove(<deletion criteria>)
```
